# Third-party native component

This connector redistributes the official ApeX Omni `zklink_sdk` native
libraries from the ApeX Protocol Python OpenAPI SDK, revision
`4d849dd84087b9551afaa55d5f37068f7ce0a103`:

https://github.com/ApeX-Protocol/apexpro-openapi

The component is used locally to create the zkLink MuSig payload signature
required by ApeX Omni. Signing seeds and private material are never sent to the
native library publisher or to another service.
